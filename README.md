## Wanntodo <a id="1" />
<hr/>

### WantoDo는 무엇인가?
* 반쪽짜리 일정관리는 이제 그만,
  WantoDo 하나로 여러분의 일상을 편하게 관리하고
  쉽게 좋은 습관들로 채워갈 수 있습니다.
### WantoDo를 만든 이유
* 투두를 등록하고 또 캘린더에 같은 일정을 등록하고..
  하루를 보람있게 살아보려고 시작한 일정 관리지만,
  반복되는 등록 작업에 지쳐가고 있지 않나요?
  한 번에 투두와 캘린더 일정을 쉽고 편리하게 관리하고
  기쁜 마음으로 보람찬 하루를 만들어가기 위해 WantoDo를 만들었습니다.
